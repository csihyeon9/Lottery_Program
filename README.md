# 🎁 경품 추첨 프로그램 (Lottery Program)

행사장에서 사용하는 **엑셀 기반 경품 추첨 프로그램**입니다.  
더블클릭 실행만으로 참가자 등록, 경품 관리, 랜덤 추첨이 가능합니다.

---

## ✅ 실행 방법 (중요)

1. `lottery_program.exe` 더블클릭
2. 프로그램 자동 실행
3. 별도 설치 필요 없음

---

## 🧩 사용 방법

### 1️⃣ 참가자 등록
- **[참가자 관리] 탭**
- 이름 / 소속 입력
- [참가자 저장] 클릭

---

### 2️⃣ 경품 확인
- **[경품 관리] 탭**
- 등록된 경품과 남은 수량 확인

---

### 3️⃣ 경품 추첨
- **[추첨] 탭**
- 경품 선택
- [추첨 시작] 클릭
- 전체화면 추첨 연출 진행
- `ESC` 키로 화면 종료

---

## 📁 데이터 저장 위치

프로그램 실행 파일과 같은 폴더에 자동 생성됩니다.


---

## ⚠️ 주의사항

- 프로그램 실행 중 강제 종료하지 마세요
- 엑셀 파일을 열어둔 상태에서 추첨하지 마세요
- `data` 폴더 삭제 시 데이터가 초기화됩니다

---

## 🔒 종료 방법

- 일반 화면: 창 닫기
- 전체화면 추첨: `ESC` 키

---

## 📌 권장 사용 환경
- Windows 10 이상
- 해상도 1920x1080 이상
