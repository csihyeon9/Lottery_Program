[경품 추첨 프로그램 사용설명서]

1. 실행 방법
- lottery_program.exe 더블클릭
- 별도 설치 필요 없음

2. 참가자 등록 (exe실행시)
- [참가자 관리] 탭 선택
- 이름 / 소속 입력
- [참가자 저장] 클릭

3. 경품 추첨
- [추첨] 탭 선택
- 경품 선택
- [추첨 시작] 클릭
- 전체화면 추첨 진행
- ESC 키를 눌러 종료

4. 데이터 저장 (폴더 내부 파일에 저장)
- 참가자 및 추첨 결과는 자동 저장됨 (data 폴더의 participants 파일 확인)

5. 주의사항
- 실행 중 강제 종료 금지
- 엑셀 파일을 열어둔 상태에서 추첨 금지
- data 폴더 삭제 시 데이터 초기화됨

6. 종료 방법
- 일반 화면: 창 닫기
- 전체화면 추첨: ESC 키
